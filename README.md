# Nhan Phan Personal Website

Static personal website for research, publications, mobile apps, video demos, expertise, and contact links.

## Files

- `index.html` - page content, SEO metadata, and structured profile data.
- `styles.css` - responsive visual design.
- `script.js` - mobile navigation, publication filtering, active navigation, reveal animation, and footer year.

## Preview

Open `index.html` in a browser. No build step or package installation is required.

## Deploy on GitHub Pages

1. Create a public repository named `Usin2705.github.io`.
2. Add these files to the repository root.
3. In GitHub, open Settings > Pages.
4. Choose deployment from the default branch and root folder.
5. Visit `https://Usin2705.github.io` after GitHub finishes publishing.

A custom domain can be added later without changing the site structure.

## Content Sources

Profile, role, contact email, ORCID, publication titles, app names, and video demo links are based on public academic profile links and details provided for the site.
